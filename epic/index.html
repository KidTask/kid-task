KidDashboard Task

KidDashboard Task is a web application aimed at task productivity for children. There is a parent-side log-in which allows parents to add kids, tasks, steps, and rewards. The kid's interface includes viewing those tasks and steps it will take to achieve a reward. This app will bring the kids task list into the digital age.
